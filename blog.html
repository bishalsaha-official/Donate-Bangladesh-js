Important FAQ

What is the Document Object Model (DOm)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.


How do you select an element from the dom?
  • document.getElementById("id")
  • document.querySelector("selector")
  • document.querySelectorAll("selector")
  • document.getElementsByClassName("class")
  • document.getElementsByTagName("tag")

What is event delegation in the context of the dom, and why is it useful?

Event delegation is a technique in the DOM where a parent element handles events for its child elements using event bubbling. Instead of attaching event listeners to multiple child elements, you attach a single listener to their parent and use event.target to determine which child triggered the event.


How do you manipulate an elements attribute and styles using the dom?

To manipulate HTML elements and their attributes using JavaScript we use the DOM (Document Object Model) function of JavaScript. The DOM represents the structure of your HTML document as a tree of objects, and you can use JavaScript to interact with these objects to change their properties, attributes, content, and more.
